Changing-Colors - is a simple Android application that changes the background color of the screen each time the user presses a button.  
The app also counts the number of button clicks (color changes) and allows the user to reset the screen to its initial state.

This project was created to practice Android development fundamentals, UI state management, and user interaction handling.

## Features
- Changes background color randomly on button click with animation
- Counts the number of color changes  
- Displays the current click counter  
- Reset button that:
  - Restores the original background color  
  - Resets the click counter to zero
 
## Technologies
- Kotlin  
- Android SDK  
- Jetpack Compose
- State Management

## How It Works
1. User presses the **Change Color** button.  
2. The background color updates.  
3. The counter increases by one.  
4. User can press **Reset** to return to the initial screen state.

## Purpose
The goal of this project was to:
- Understand event handling in Android  
- Work with UI updates and state changes  
- Practice basic application structure

## How to Run
1. Clone the repository  
2. Open the project in Android Studio  
3. Run the application on an emulator or physical device
